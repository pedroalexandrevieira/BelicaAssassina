# BelicaAssassina

BelicaAssassina is a challenge by Academia de Código bootcamp.
The goal is to develop a group game in 10 days.

**Technologies**

`Java` `Simple Graphics`

**Commands**

`Press enter to start the game`

`Press space to shoot`
